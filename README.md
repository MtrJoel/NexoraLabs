# 🚀 NexoraLabs - Landing Page

Sitio web oficial de **NexoraLabs**, enfocado en ofrecer soluciones digitales modernas como desarrollo de software a medida, páginas web y servicios de mantenimiento.

---

## 🌐 Descripción

Esta landing page fue diseñada con un enfoque moderno, minimalista y orientado a conversión, destacando los servicios principales de la marca:

* Desarrollo de sistemas personalizados
* Creación de páginas web
* Mantenimiento y soporte técnico
* Implementación de mejoras

El objetivo principal es presentar la propuesta de valor de NexoraLabs de forma clara, profesional y atractiva.

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Diseño responsive
* Animaciones con Intersection Observer

---

## ⚡ Características

* Diseño moderno tipo landing page
* Animaciones suaves al hacer scroll
* Estructura optimizada para conversión
* Compatible con dispositivos móviles
* Código limpio y fácil de escalar
